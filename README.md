# Melee101Launcher
A accessable &amp; open-source launcher for Windows made in Python for Melee101.

## Explanation:
When Melee101 comes out, instead of just getting the base game, you have the choice to choose between the launcher or the base game. The Launcher would have automatic updates, custom themes, and would allow you to see your game progress and login through the launcher instead of the game.

## Why:
Well, not only would it increase our Python app development but, it would also help people who have **ZERO** idea on how to play the game.
